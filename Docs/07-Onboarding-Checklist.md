# 7) Onboarding Checklist

## Kurulum / İlk erişim

- [ ] Uygulama çalışıyor ve giriş ekranı açılıyor
- [ ] Kullanıcı hesabı ile **Giriş Yap** doğrulandı

## Veri hazırlığı

- [ ] Gerekirse **Örnek Excel’i İndir** ile şablon alındı
- [ ] İlk veri girişi için Excel dosyası hazırlandı
- [ ] **Excel Dosyası Yükle** ile içe aktarma test edildi

## Temel doğrulamalar

- [ ] **Öğrenciler** listesi görüntüleniyor
- [ ] **Üyeler** listesi görüntüleniyor
- [ ] **Köyler** listesi görüntüleniyor
- [ ] **Yardımlar** listesi görüntüleniyor
- [ ] **Toplantılar** ekranında **Yeni Toplantı Oluştur** ile test toplantısı açıldı
- [ ] **Aktivite Logları** ekranında **Filtrele** ile kayıtlar görüldü

## Ayarlar

- [ ] **Sektörler/Üniversiteler/Bölümler/Dönemler/Burs Tutarları** listeleri kontrol edildi
- [ ] Gerekli temel değerler girildi
